# Proyecto de Simulación para "Sistemas de Tiempo Real"
Trabajo integrador para la materia Sistemas de Tiempo Real (I115) de la carrera "Ing. en Computación" (FI-UNLP, plan 2011).

Simulación de comportamiento de la estación de trenes de Berazategui. Reimplementación en GAMA de un proyecto de 2022 implementado en NetLogo, para comparación de rendimiento entre ambas herramientas.

Versión Final (Diciembre 2023).

# Herramientas requeridas

Para la ejecución del proyecto se requiere del software [GAMA](https://gama-platform.org/). El proyecto del 2022 puede ejecutarse con [NetLogo](https://ccl.northwestern.edu/netlogo/). Se recomienda un equipo relativamente "nuevo" (CPU de 64 bits multicore, con memoria mayor a 4GiB de RAM) para ejecutar el proyecto de GAMA cómodamente. En el equipo usado para pruebas (Intel Core i3 "Sandy Bridge", 16GiB RAM) la simulación se ejecuta correctamente, aunque muy lentamente si se activa el trazado del gráfico en tiempo real. Sin el gráfico, la simulación se ejecuta a velocidad razonable.
